Install on Chrome, Edge, or Brave

1

Download the extension

Get aid-chrome.zip from the latest release and unzip it.

2

Open Extensions page

Navigate to chrome://extensions (or edge://extensions / brave://extensions)

3

Enable Developer Mode

Toggle the switch in the top-right corner.

4

Load the extension

Click Load unpacked and select the unzipped aid-chrome folder.

5

Pin it

Click the puzzle icon in the toolbar and pin AID for quick access.

Install on Firefox

1

Download the extension

Get aid-firefox.xpi from the latest release.

2

Install the add-on

Open about:addons, click the gear icon, select Install Add-on From File, and choose the .xpi file.

3

That's it

The AID icon appears in your toolbar. Click it to scan any page.

Build from source

1

Clone the repository

git clone https://github.com/KillAllTheHippies/aid.git && cd aid && git checkout aid-extension

2

Build

Bash: cd aid-extension && ./build.sh all
PowerShell: cd aid-extension; .\build.ps1

3

Load the built extension

Follow the Chrome or Firefox steps above, pointing to aid-extension/dist/chrome/ or installing the .xpi.
